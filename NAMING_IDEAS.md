## GitHub Naming Best Practices

✅ **Do:**
- Use lowercase
- Use hyphens for multi-word names
- Keep it short (2-3 words max)
- Make it memorable
- Check availability first

❌ **Don't:**
- Use underscores (harder to type)
- Make it too long
- Use special characters
- Use numbers unless necessary

## Quick Check Commands
